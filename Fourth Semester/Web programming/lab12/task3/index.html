CSS

Спецификация CSS(Cascading Style Sheets) позволяет остаться в рамках декларативного характера разметки и дает полный контроль над формой представления элементов HTML-разметки.

Каскадные таблицы стилей призваны резрешить противоречие между точностью определения размеров картинок и приложений с одной стороны,и точностью определения размеров блоков текстов и его начертания - с другой.
